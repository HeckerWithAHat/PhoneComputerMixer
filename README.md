## PhoneComputerMixer: A custom-built virtual fader bank that automatically updates to include all active apps.

## The Problem

With an increasing amount of apps running concurrently on my laptop, just the volume button on my laptop was not enough. The Windows volume mixer was a small window that became another app that was controlled by my mouse and annoying to use.

PhoneComputerMixer tackles this issue head-on by providing a simple web app that can be accessed on the local network by a phone or tablet. It contains faders for every open window, allowing easy and natural control.

## Features

- **Fader Bank Autoscaling** Automatically scales the faders and website to fit on any device.
- **App Discovery** This project automatically discovers when a new app runs and adds it as a fader.
- **Icon Support** The web app pulls the icon of the app to make it easier to know which app you are controlling.

## Steps to Use DependGuard

1. Download the source code
2. Run main.py
3. Visit the site on any device
4. Slide the fader up to increase the volume and down to decrease.
